## O que é necessário para utilizar o Appium?

- [x] Node.js
- [x] Appium
- [x] Android Studio (opcional)
- [x] Android SDK
- [x] Java SDK
- [x] Appium Doctor (opcional)
- [x] Uiautomator2
- [x] Appium-Python-Client
